Étape 1 : Modification du HTML

Vous devez utiliser la directive *ngIf d'Angular pour afficher des éléments conditionnellement. Dans votre cas, vous voulez afficher soit la localisation normale (si elle est présente), soit le texte "pas en sortie" (si la localisation est absente).

html
<ion-item *ngFor="let contact of filteredContacts">
  <!-- ... autres éléments ... -->
  <ion-label>
    <h2>{{ contact.nom }}</h2>
    <h3 *ngIf="contact.localisation; else noLocation">
      <i class="bi bi-geo-fill" style="margin-right: 4px; color: rgb(255, 0, 72);"></i>{{ contact.localisation }}

Dans cet extrait :

  • *ngIf="contact.localisation" vérifie si la propriété localisation existe et est non vide.
